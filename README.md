## :wrench: Install and run

Open terminal:

```sh
# Clone this repo
git clone https://github.com/osvaldokalvaitir/nodejs-rocketbox

# Entry in folder
cd nodejs-rocketbox

# Install deps with npm or yarn
npm install | yarn

# Launch the app with npm or yarn
npm dev | yarn dev
```
